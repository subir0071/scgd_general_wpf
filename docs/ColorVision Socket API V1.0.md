## ColorVision Socket API V1.0

### Base URLs

- **Testing Environment:** `http://127.0.0.1:6666`

### Standard Unhandled Return

- **Response:** `Unhandled Function Call`

### API Endpoints

#### 运行流程 (Flow)

- **Endpoint Format:** `Flow,{流程名}`
- **Example:** `Flow,poitest`
- **Successful Response:**
  - `Run {流程}`
- **Error Response:**
  - `Cant Find Flow {流程} 优化文档`
